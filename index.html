Bomberman

This is a project I created using only HTML, CSS and javascript. I made this with a team of two others from 01Founders, and the idea was mostly to recreate the classic game bomberman.

Flappy Fish

This is a game I made with html, css and javascript following a tutorial. The controls are click to go up and shift to boost. the bar on the left shows how much boost energy you have left.

Multiplayer Snake

This is another game i made following a tutorial. You can have two players playing on the same keyboard, one with the WASD keys and one with the arrow keys. There is currently a bug where the mushroom you eat sometimes spawns out of bounds and softlocks you.
